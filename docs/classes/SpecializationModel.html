Modèle de gestion des spécialisations médicales

Cette classe gère les opérations relatives aux spécialisations médicales dans l'application MedBoard. Elle permet de vérifier l'existence, récupérer la liste complète et obtenir les paires clé-valeur pour les formulaires de sélection.
